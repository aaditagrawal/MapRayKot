# MapRayKot

A quiet little atlas game. Two ways to learn the borders you've forgotten.

Dedicated to my dearest friends who I've burnt many geoguessr hours with - Rithwik Rayani and Balleswar Kota.

## The game

**Locate** — We name a country. You drop a pin on the world map. The closer your pin lands to the real border, the higher the score. A perfect hit inside the border is 1000 points; points fall off as you drift further away.

**Name** — A country lights up with a close-up inset. You type its name. Four points for a hit, minus one for a skip. Keep going until the timer runs out.

Each run is saved locally so you can watch your ladder climb over time. No accounts, no servers, no leaderboards - just you and the map.

(might do multiplayer in the future though, for funsies)

## Run it

```bash
bun install
bun run dev
```

Then open [localhost:3000](http://localhost:3000).
